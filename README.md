| 时间管理助手 | time-manager | 直接使用当时具体的prompt：帮我创建一个skill用于我的时间管理助手(time-manager），其主要功能为：1、以“开启新的一天，进行时间规划”为启动提示词，或者类似即可；2、排查可能之前的日程安排文件（文本文件）、个人信息文件（包括身份等），这里使用代码或者其他的mcp等形式读取文件（文件目录在该skill下），并以代码等形式调用查询当前日期和时间；3、然后告知之前安排工作的剩余工作量，并询问是否需要添加一些新的任务，并且提供大概的工作量和时间截止要求；4、在过程中提取个人信息，并将新的任务更新在文件当中；5、询问今天的日程安排的要求（比如有没有其他突发事件占据时间，今天以简易的任务还是复杂的任务量等），并排出一个以时刻为标准的时间表；6、询问是否微调，将新的工作表更新在文本文件当中存档，开始工作；7、若以“今天已经结束，进行时间管理的总结”为提示词，读取相关文件，调用查询当前日期时间。按照今天的时间表询问每个任务的完成情况，然后更新中的任务的文本文件。文本保存：1、个人信息文件；2、每日时刻表（以具体日期命名的一系列文件）；3、任务总表文件（包含完成、未开始、进行中等状态的任务） | 2026.1.14 |
